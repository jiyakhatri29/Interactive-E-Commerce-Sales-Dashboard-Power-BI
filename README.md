# Interactive-E-Commerce-Sales-Dashboard-Power-BI
Built an interactive Power BI dashboard to track sales, profit, orders, customer preferences, payment modes, and regional performance. The dashboard includes KPI cards, trend analysis, category insights, and dynamic filters for better business decision-making.

## Key Features
- KPI Cards (Total Sales, Profit, Orders)
- Quarterly Filtering
- State-wise Analysis
- Category-wise Analysis
- Payment Mode Insights
- Monthly Profit Trend

## Tools Used
- Power BI
- DAX
- Data Visualization
